# Quiz das Pombagiras - O Mistério das Giras

Um quiz denso, atmosférico e bem macabro sobre as rainhas da encruzilhada, com estética gótica pesada, velas tremulando no código e pétalas de rosa caindo eternamente na tela.

https://user-images.githubusercontent.com/999999999/999999999-sample.gif  
*(se tiver gif/vídeo de demonstração, colocar aqui depois)*

## Características principais

- 15 perguntas densas sobre as Pombagiras mais conhecidas
- Sistema de pontuação que revela sua **Pombagira Rainha** no final
- Design extremamente gótico: sangue, velas, caveiras, sombras, pétalas caindo
- Animações suaves de transição entre perguntas
- Feedback visual e sonoro (bem discreto) a cada resposta
- Contador de visitas persistente (local + Supabase)
- Total imersão na gira: cores escuras, fontes serifadas antigas e detalhes macabros

## Tecnologias utilizadas

- HTML5 + CSS3 (muito CSS mesmo...)
- Tailwind CSS (v2)
- Font Awesome 6
- Google Fonts (Cinzel + Gothic A1)
- JavaScript puro (sem frameworks pesados)
- Supabase (opcional - contador de visitas)
- Animações CSS + transições suaves
- Áudio procedural bem creepy (oscillator)
